This project is a pet grooming salon operating system for DAZUL.

## Core rules
- Mobile-first and tablet-friendly UI
- Korean language UI
- Warm, premium tone
- Simple screens for busy staff
- Minimal typing
- No medical claims
- Focus on grooming records, care notes, recommendations, and guardian reports

## UX rules
- Large buttons
- Fast input
- One-screen workflow when possible
- Staff should be able to finish a record quickly

## Initial build priority
1. Grooming record form
2. Record list
3. Guardian report page
4. History page

## Avoid
- Complex dashboard first
- Photo upload system first
- Overly technical UI
- Long text input
